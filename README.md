# ToDo-App

A straightforward task manager. 
Creates, displays, updates, and deletes tasks. 
Stores all data in the browser.

## Features
- create tasks
- edit tasks
- mark as completed
- delete tasks
- filters: all, completed, active
- stores data in localStorage


## Installation and Run
``` 
    git clone https://github.com/tblackwood60/ToDo-App.git
    cd ToDo-App
    npm install
    npm start
```

## Technologies
- HTML
- CSS
- JavaScript
- Animate.css
